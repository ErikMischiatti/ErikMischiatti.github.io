# Erik Mischiatti - Portfolio

Personal portfolio website for Erik Mischiatti, showcasing robotics, electronics, autonomous systems, and mechatronics projects.

## Stack

- Static HTML
- CSS
- JavaScript
- GitHub Pages

## Project Structure

- `index.html` - home page and main portfolio sections
- `pages/` - individual project pages
- `assets/css/` - template and custom stylesheets
- `assets/js/` - static JavaScript used by the template
- `pics/` - images, logos, GIFs, and videos
- `docs/` - CV, reports, papers, and thesis PDFs

## Local Preview

Run a static server from the repository root:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000/`.

## Deploy

GitHub Pages publishes the site from the configured branch of the `ErikMischiatti.github.io` repository.

## Maintenance

- Check internal links when adding or renaming pages and assets.
- Optimize large images, GIFs, and videos when updating project media.
- Keep the CV and project pages aligned with current portfolio content.

Master Thesis Project: Autonomous Robot Learning

Project Overview

This project focuses on teaching a robot to learn and reverse complex tasks using behavior trees and 6D Pose Estimation.

Technologies Used

  • Behavior Trees
  • 6D Pose Estimation
  • Autonomous Learning Algorithms

Results

The robot successfully learned complex tasks and was able to reverse them efficiently.

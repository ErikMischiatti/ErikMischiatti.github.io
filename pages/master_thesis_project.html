This project focuses on teaching a robot to learn and reverse complex tasks using behavior trees and object detection.

Bidirectional Invariant Representation (DHB)

The Bidirectional Invariant Representation (BIR) is a robust framework for describing rigid body movements invariant to rotations, translations, scaling, and time variations. This framework is particularly effective in gesture recognition and robotic motion reproduction, where robustness against variations is crucial.

In my work, the DHB convention is applied to reconstruct motion trajectories by mapping Cartesian coordinates into an invariant space. This method ensures that motion can be reproduced with high fidelity, regardless of variations in the environment, speed, or orientation.

In essence, DHB allows the robot to generalize the learned behavior and apply it to new scenarios. This is particularly useful for tasks like cable insertion, which requires precision and adaptability.

Setup and Configuration of the Environment

The experimental environment includes the assembly board system and the robotic arm system. The board is equipped with Aruco markers to define its position, and 3D-printed components for the manipulation tasks.

The Franka Emika Panda robotic arm, with a mounted Intel RealSense D435i camera, is used to manipulate objects with high precision, performing tasks like grasping and moving cables.

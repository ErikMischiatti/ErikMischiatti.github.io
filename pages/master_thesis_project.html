Master Thesis Project - Autonomous Robot Learning

This project aims to teach a robot to autonomously learn and reverse complex tasks using Behavior Trees (BT), Learning from Demonstration (LfD), and 6D Pose Estimation.

Key Concepts

Behavior Trees (BT): A modular and scalable control architecture widely used in robotics for decision-making and execution of actions. BTs allow flexible and hierarchical task handling, which is critical for autonomous systems.

Learning from Demonstration (LfD): A technique where robots learn tasks by observing and imitating the actions demonstrated by humans. This is particularly useful for robotic assembly tasks that require flexibility and precision.

Dynamic Movement Primitives (DMP): A robust framework that allows robots to encode and reproduce smooth, nonlinear trajectories learned from demonstrations. DMPs ensure global asymptotic stability and adaptability in dynamic environments.

Materials and Equipment

Robotic Arm: Franka Emika Panda, a high-precision manipulator with 7 degrees of freedom, integrated torque sensors, and ROS compatibility.

Camera: Intel RealSense D435i, providing 3D vision and depth perception for accurate object tracking.

Custom Board: A modular surface for robotic manipulation, inspired by NIST, equipped with Aruco markers for precise tracking.

System Architecture

The system consists of a robot manipulator, camera, and modular board for manipulation tasks. The Franka Emika Panda robot interacts with objects on the board, while the Intel RealSense D435i camera provides real-time depth and object recognition data. ROS (Robot Operating System) coordinates the actions.

Implementation

The robot learns to perform tasks via Learning from Demonstration. Through teleoperation or kinesthetic teaching, the robot records demonstrated actions, extracts key features, and uses these to replicate the task autonomously. DMPs enable the robot to adapt movements to new goals while maintaining stability.

Results and Future Development

The system is currently under active development. Initial tests show promising results in task generalization and adaptability. Future improvements will focus on refining the precision of object manipulation and enhancing the robustness of the learning algorithms.

Green Vigilance: Drone Innovation for Early Crop Disease Detection

Integrating UAVs and UGVs for precision agriculture to detect plant diseases early and improve resource efficiency.

Status: Ongoing...

Project Overview

This project develops a distributed UAV-UGV system aimed at enhancing early crop disease detection. By integrating aerial surveys with ground-level inspections, the system provides detailed spatial and health data of agricultural fields. Advanced algorithms like Weighted Least Squares (WLS) and Extended Kalman Filter (EKF) enhance navigation and decision-making under uncertain conditions.

System Architecture

The architecture integrates UAVs for wide-area aerial surveys and UGVs for detailed inspections. UAVs use hybrid depth sensors to create 3D maps, guiding UGVs to targeted areas for precision monitoring and interventions.

Methodologies

Key methodologies include:

  • Hybrid Depth Sensing: Collects 3D spatial data for comprehensive field mapping.
  • Weighted Least Squares (WLS): Enhances UGV positioning accuracy by aggregating reliable position estimates.
  • Extended Kalman Filter (EKF): Refines UAV navigation by integrating GPS and IMU data.
  • Heatmap Generation: Identifies high-probability disease zones, guiding UGV operations effectively.
